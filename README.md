## Local Development

Just open `public/index.html` in your browser. That's it.

## Deployment

This project is configured for [Render.com](https://render.com):

1. Push to GitHub
2. Connect repo to Render
3. Render auto-detects settings from `render.yaml`
4. Live in ~2 minutes
